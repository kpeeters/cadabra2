Download and install

Ubuntu/Mint/Debian

If you are on an Ubuntu/Mint/Debian system, you can install a recent version of Cadabra by enabling the PPA,

sudo add-apt-repository ppa:kasper-peeters/ppa
sudo apt-get update
sudo apt-get install cadabra2

You can then start the notebook interface with

cadabra-gtk

or pick it from the 'Education' menu.

Build from source (Linux or OS X)

Alternatively, you can compile Cadabra from source yourself (not difficult). The source is distributed via github, see the kpeeters/cadabra2 repository for more details, including the pre-requisites.
